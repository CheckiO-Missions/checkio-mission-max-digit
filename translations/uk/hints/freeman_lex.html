З якого інструменту мені слід розпочати?
Використай вбудовану функцію str(), щоб конвертувати вхідне число в рядок. 

a = 12
a = str(a)
print(a)

# 12 виглядає так само, але тепер це рядок :-)
Зроблено. Але не розумію, як продовжити.
Наступний крок, це використати вбудовану функцію map(), яка застосовує визначену функцію до кожного елементу послідовності. У цьому випадку, тобі потрібно конвертувати кожен символ створеного рядка назад в ціле число. 

a = map(int, str(a))
Результатом є об'єкт map. Якщо конвертувати його в список, ти побачиш: 

print(list(a))

# output:
# [1, 2]
Як обрати найбільший елемент з цього списку?
Використай вбудовану функцію max(), щоб знайти найбільший елемент у послідовності. Ця функція може приймати об'єкт map object як аргумент. Немає потреби в перетворенні в список. 

a = max(map(int, str(a)))
print(a)

# output:
# 2
То як має виглядати мій остаточний розв'язок?
Тобі лишилося обгорнути код в вираз функції і повернути результат функції max(). Ось код: 

def max_digit(a: int) -> int:

    return max(map(int, str(a)))
Ця функції проста і коротка, тож ти можеш використати lambda вираз (анонімну функцію), щоб записувати такі функції більш компактно. 

max_digit = lambda a: max(map(int, str(a)))

Цей розв'язок використовує навіть менше функцій, але і є менш очевидним. Коли ти застосовуєш max() до рядка без визначеного ключа сортування, символи сортуються в лексикографічному порядку, де символи від "0" до "9" відсортовані так само як цифри від 0 до 9. 

max_digit = lambda number: int(max(str(a)))
Якщо у тебе лишилися сумніви, ось детальний покроковий розв'язок. 

def max_digit(a: int) -> int:

    current_max = 0
    for num in str(a):
        num = int(num)
        if num > current_max:
            current_max = num

    return current_max
