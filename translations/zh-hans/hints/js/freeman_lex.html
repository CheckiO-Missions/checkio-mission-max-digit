一般的想法是将数字 "分解 "成不同的数字，然后选择其中最大的数字。

好吧，我应该从什么开始？

如果更具体一点，你应该把数字转换成字符串，把独立的字符组成一个数组，把每个字符再转换成数字，然后取最大的一个。

用String() 方法将数字转换成字符串。 

String(value)

如何将这个字符串做成一个独立的字符数组？

使用string.split() 方法，其中第一个参数是空字符串。 

String(value).split("")

如何将数组中的每个元素（字符串类型）转换回数字类型？

使用带有函数参数的array.map() 方法，你需要为每个元素调用该方法。在这个例子中，它是Number 方法。 

String(value).split("").map(Number)

好了，接下来该怎么做？

要找到这些数字中的最大值，你需要把它们作为Math.max() 方法的参数，像这样。 

Math.max(   String(value).split("").map(Number))

但这里有一个技巧。这些数字现在被装在数组中。但是你需要用逗号分隔的数字作为参数来调用Math.max()方法。所以要用spread 运算符来解包数组。 

Math.max(...String(value).split("").map(Number));

那么，我的最终解决方案应该是什么样子的呢？

你只需要将你的代码包装成函数表达式，并返回你所建立的表达式。

下面是整个解决方案：

function maxDigit(value: number): number {

    return Math.max(...String(value).split("").map(Number));
    }

这个解决方案比较短。正如你所看到的，它对字符（类型string ）进行排序，而不是数字。作为字符串的一位数的词法顺序（"0"... "9"）与作为数字的数学顺序（0...9）是一样的。

请注意，在第一个变体中，返回的是string 类型的结果。这是因为Javascript认为，5 == "5" ，例如。在第二个变体中，你用+ 将字符串转换成数字。 

function maxDigit(value: number): number {

    return Math.max(...value.toString());
    // or
    return +Math.max(...value.toString());
}

这里是不转换为字符串的逐步解决方案。你可以通过用数字除以10并取其余数来得到分离的数字。 

function maxDigit(value: number): number {

    let res = 0;
    while (value>= 1) {
        res = Math.max(res, value%10);
        value = Math.floor(value/10);
    }
    return res;
}
