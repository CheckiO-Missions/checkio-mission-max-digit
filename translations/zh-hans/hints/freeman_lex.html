一般的想法是将数 "分解 "成不同的数，然后选择其中最大的数。

我应该从什么仪器开始？

使用内置的str() 函数，将输入的数字转换成字符串。 

a = 12
a = str(a)
print(a)

输出：

12

12 看起来是一样的，但现在是字符串:-)

完成了。但不知道如何继续。

下一步是使用内置的map() 函数，它对序列中的每个元素都实现了特定的功能。在这种情况下，你需要将创建的字符串的每个字符转换为整数。 

a = map(int, str(a))

其结果是map对象。如果你把它转换为列表，你会看到： 

print(list(a))

输出：

[1, 2]

如何选择列表中的最大元素？

使用内置的max() 函数来寻找序列中的最大元素。这个函数可以接受map对象作为其参数。不需要转换为列表。 

a = max(map(int, str(a)))
print(a)

输出：

2

那么，我的最终解决方案应该是什么样子的呢？

你只需要将代码包装成函数表达式，并返回max() 函数的结果。

以下是整个代码：

def max_digit(a: int) -> int:

    return max(map(int, str(a)))

这个解决方案很简单，也很短，所以你可以使用lambda表达式（匿名函数）来把这类函数写得更紧凑。 

max_digit = lambda a: max(map(int, str(a)))

这个解决方案使用的函数更少，但不太明显。当你实现max() 到字符串时，没有特定的排序键，字符是以词汇顺序排序的，其中从 "0 "到 "9 "的字符与从0到9的数字排序方式相同。 

max_digit = lambda a: int(max(str(a)))

如果你有任何疑问，这里有详细的分步解决方案。 

def max_digit(a: int) -> int:

    current_max = 0
    for num in str(a):
        num = int(num)
        if num> current_max:
            current_max = num

    return current_max
