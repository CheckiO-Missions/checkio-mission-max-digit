Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

ogólny pomysł polega na "rozbiciu" liczby na osobne cyfry, a następnie wybraniu największej z nich.

Od jakiego instrumentu powinienem zacząć?

Proszę użyć wbudowanej funkcji str(), aby przekonwertować liczbę wejściową na ciąg znaków. 

a = 12
a = str(a)
print(a)

Dane wyjściowe:

12

12 wygląda tak samo, ale jest teraz ciągiem znaków:-)

Gotowe. Ale nie wiem, jak kontynuować.

Następnym krokiem jest użycie wbudowanej funkcji map(), która implementuje określoną funkcję do każdego elementu sekwencji. W tym przypadku należy przekonwertować każdy znak utworzonego ciągu z powrotem na liczbę całkowitą. 

a = map(int, str(a))

Wynikiem jest obiekt mapy. Jeśli przekonwertuje go Pan na listę, zobaczy Pan: 

print(list(a))

Wynik:

[1, 2]

Jak wybrać maksymalny element na liście?

Proszę użyć wbudowanej funkcji max(), aby znaleźć maksymalny element w sekwencji. Funkcja ta może przyjąć jako argument obiekt mapy. Nie trzeba konwertować na listę. 

a = max(map(int, str(a)))
print(a)

Wynik:

2

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcji i zwrócić wynik funkcji max().

Oto cały kod:

def max_digit(a: int) -> int:

    return max(map(int, str(a)))

To rozwiązanie jest proste i krótkie, więc można użyć wyrażenia lambda (funkcji anonimowej), aby napisać takie funkcje bardziej kompaktowo. 

max_digit = lambda a: max(map(int, str(a)))

To rozwiązanie wykorzystuje jeszcze mniej funkcji, ale jest mniej oczywiste. Kiedy zaimplementują Państwo max() to string bez określonego klucza sortowania, znaki są sortowane w porządku leksykograficznym, gdzie znaki od "0" do "9" są sortowane tak samo jak cyfry od 0 do 9. 

max_digit = lambda a: int(max(str(a)))

Jeśli mają Państwo jakiekolwiek wątpliwości, poniżej znajduje się szczegółowe rozwiązanie krok po kroku. 

def max_digit(a: int) -> int:

    current_max = 0
    for num in str(a):
        num = int(num)
        if num> current_max:
            current_max = num

    return current_max
