Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

ogólna idea polega na "rozbiciu" liczby na poszczególne cyfry, a następnie wybraniu największej z nich.

Ok, od czego powinienem zacząć?

Mówiąc dokładniej, powinien Pan przekonwertować liczbę na ciąg znaków, utworzyć tablicę oddzielnych znaków, przekonwertować każdy znak z powrotem na liczbę, a następnie wybrać największy.

Proszę przekonwertować liczbę na ciąg znaków za pomocą metody String(). 

String(value)

Jak utworzyć tablicę oddzielnych znaków z tego ciągu?

Proszę użyć metody string.split(), gdzie pierwszym argumentem jest pusty ciąg znaków. 

String(value).split("")

Jak przekonwertować każdy element (typu String) tablicy z powrotem na typ Number?

Proszę użyć metody array.map() z argumentem funkcji, którą należy wywołać dla każdego elementu. W tym przypadku jest to metoda Number. 

String(value).split("").map(Number)

Ok, co robić dalej?

Aby znaleźć maksimum wśród tych cyfr, należy użyć ich jako argumentów metody Math.max(), jak poniżej. 

Math.max(   String(value).split("").map(Number))

Ale jest tu pewna sztuczka. Te cyfry są teraz spakowane wewnątrz tablicy. Należy jednak wywołać metodę Math.max() z liczbami oddzielonymi przecinkami jako argumentami. Proszę więc rozpakować tablicę za pomocą operatora spread. 

Math.max(...String(value).split("").map(Number));

Jak powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcji i zwrócić zbudowane wyrażenie.

Oto całe rozwiązanie:

function maxDigit(value: number): number {

    return Math.max(...String(value).split("").map(Number));
    }

To rozwiązanie jest krótsze. Jak widać, sortuje ono znaki (typu string), a nie liczby. Kolejność leksykograficzna jednocyfrowych liczb jako ciągów ("0"... "9") jest taka sama jak matematyczna kolejność ich jako liczb (0...9).

Proszę zauważyć, że w pierwszym wariancie zwracany jest wynik typu string. Dzieje się tak dlatego, że Javascript bierze pod uwagę, że 5 == "5", na przykład. W drugim wariancie konwertują Państwo ciąg znaków na liczbę za pomocą +. 

function maxDigit(value: number): number {

    return Math.max(...value.toString());
    // or
    return +Math.max(...value.toString());
}

Oto rozwiązanie krok po kroku bez konwersji na ciąg znaków. Oddzielone cyfry uzyskuje się dzieląc liczbę przez 10 i pobierając resztę. 

function maxDigit(value: number): number {

    let res = 0;
    while (value>= 1) {
        res = Math.max(res, value%10);
        value = Math.floor(value/10);
    }
    return res;
}
