üí° The general idea is to "break" number into separate digits and then choose the biggest among them.

Ok, what should I start from?

If to be more specific, you should convert number into string, make an Array of separate characters, convert each character back into number and then take the biggest.

Convert the number into string with String() method. 

String(value)

How to make an Array of separate characters from this string?

Use the string.split() method, where first argument is empty string. 

String(value).split("")

How to convert each element (type String) of the Array back to type Number?

Use array.map() method with argument of function, which you need to call for every element. In this case, it's Number method. 

String(value).split("").map(Number)

Ok, what to do next?

To find the maximum among these digits you need to use them as arguments of Math.max() method, like this. 

Math.max(   String(value).split("").map(Number))

But there is a trick here. These digits are packed inside Array now. But you need to call Math.max() method with comma separated numbers as arguments. So unpack Array with spread operator. 

Math.max(...String(value).split("").map(Number));

So, how my final solution should look like?

You just need to wrap your code into function expression and return the expression you've built.

Here is the whole solution:

function maxDigit(value: number): number {

    return Math.max(...String(value).split("").map(Number));
    }

This solution is shorter. As you may see, it sorts characters (type string), not numbers. Lexicographic order of one-digit numbers as strings ("0"..."9") is the same as mathematical order of them as numbers (0...9).

Note, that in the first variant result of type string is returned. It's because Javascript considers, that 5 == "5", for example. In the second variant you convert string to number with +. 

function maxDigit(value: number): number {

    return Math.max(...value.toString());
    // or
    return +Math.max(...value.toString());
}

Here is step-by-step solution without converting to string. You get separated digits by dividing number by 10 and taking the reminder. 

function maxDigit(value: number): number {

    let res = 0;
    while (value >= 1) {
        res = Math.max(res, value%10);
        value = Math.floor(value/10);
    }
    return res;
}
