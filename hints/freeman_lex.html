The general idea is to "break" number into separate digits and then choose the biggest among them.
What instrument should I start from?
Use build-in str() function to convert input number into string. 

    a = 12
    a = str(a)
    print(a)
    
    # output:
    # 12 looks the same, but it's string now :-)
Done. But don't know how to proceed.
The next step is to use build-in map() function, which implements specific function to each element of sequence. In this case, you need to convert each character of created string back into integer. 

    a = map(int, str(a))
The result is map object. If you convert it into list, you see: 

    print(list(a))
    
    # output:
    # [1, 2]
How to choose maximum element in the list?
Use build-in max() function to find maximum element in sequence. This function can take map object as its argument. No need to convert into list. 

    a = max(map(int, str(a)))
    print(a)
    
    # output:
    # 2
So, how my final solution should look like?
You just need to wrap code into function expression and return result of max() function. Here is the code: 

    def max_digit(a: int) -> int:
    
        return max(map(int, str(a)))
This function is simple and short so you may use lambda expression (anonymous function) to write such functions more compact. 

    max_digit = lambda a: max(map(int, str(a)))

This solution uses even less functions, but is less obvious. When you implement max() to string without specific sorting key, characters are sorted with lexicographical order, where characters from "0" to "9" are sorted the same way as digits from 0 to 9. 

    max_digit = lambda a: int(max(str(a)))
If you have any doubts, here is detailed step-by-step solution. 

    def max_digit(a: int) -> int:
    
        current_max = 0
        for num in str(a):
            num = int(num)
            if num > current_max:
                current_max = num
    
        return current_max
